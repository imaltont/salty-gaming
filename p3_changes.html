General changes

While the general changes from design to finished product are not that visible, some changes were made. We decided to cut and remove some of the javascript as described below, and added more to fill the gap and make the page more user friendly. Minor changes to the pictures all over the web page as we had to make/find suitable pictures. The names of most of the pictures has also changed. We added the form after the shop page so the costumer had an easier time buying t-shirts. Also we do have some minor differences to the layout on the contact.html due to differences in CSS. We also dropped the hover item information in JS and went with a simpler approach. The logo is made by Espen in Photoshop CS6.

The group decided that a dedicated site map was redundant, considering you can go from anywhere on the web page to anywhere else, just using the navigation bar. The navigation bar in this case therefore works as the site map, making a dedicated site map not necessary. The feedback for navigation on the page from the testing also confirmed that a site map was not necessary.

Modularity

The modularity requirment was met by making a javascript file, header_footer.html, that makes the header and footer for all pages. This script inserts the header and footer into all pages, except the index page. The index page only gets the footer, as the header is made differently here. It was decided that the sidebars should not be made by javascript, as the content of them is not the same on all pages, some are just links, som uses buttons, and the links are not pointing to the same sites (player profiles).
